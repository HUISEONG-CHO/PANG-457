# Mission 1 — 후지산 (Mt. Fuji, Japan)

## 개요

팡의 첫 번째 미션으로, **일본 후지산**을 배경으로 한다.  
게임의 기본 메커니즘(풍선 분열, 작살 조작)을 익히는 입문 구간이다.

---

## 스테이지 구성

| 스테이지 | 시간대 | 특징 |
|----------|--------|------|
| Stage 1 | 아침 | 대형 풍선 1개, 가장 단순한 구성 |
| Stage 2 | 오후 | 대형 풍선 2개, 소폭 난이도 증가 |
| Stage 3 | 밤 | 풍선 수 증가 또는 속도 증가 |

- 3개 스테이지 모두 **동일한 후지산 배경**, 시간대만 변경
- 지형은 **평탄한 단층 구조** — 사다리·발판 없음

---

## 난이도

### 풍선 등장
- 대형(1단계) 풍선으로 시작 → 격파 시 중형 2개로 분열 → 소형 → 극소
- Stage 1 기준 최대 분열 시 **8개의 극소 풍선** 동시 존재 가능

### 난이도 특징
- 장애물 없는 평탄한 지형으로 풍선 회피 공간이 넓음
- 풍선 이동 속도가 느려 초보자도 패턴 파악 용이
- 스테이지가 진행될수록 풍선 수·속도 소폭 상승

### 주의 포인트
- 대형 풍선을 낮은 위치에서 격파할 경우, 분열된 풍선이 화면 하단에 밀집해 회피 난이도 급상승
- 작살 발사 후 와이어가 걷히기 전까지 추가 발사 불가 → 타이밍 관리 필요

---

## 클리어 조건

- 3개 스테이지 전부 클리어 → Mission 2 (다음 지역)로 진행
