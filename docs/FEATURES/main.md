# 메인 화면 (Main Screen)

## 개요

게임 진입 시 가장 먼저 보이는 화면으로, 게임 시작 및 기본 탐색을 제공한다.

---

## 화면 구성 요소

```
┌─────────────────────────────────────┐
│                                     │
│            🎮 PANG                  │  ← 게임 타이틀
│                                     │
│         [ GAME START ]              │  ← 시작 버튼
│                                     │
│                                     │
└─────────────────────────────────────┘
```

### 1. 게임 타이틀
- 화면 상단 중앙에 **"PANG"** 텍스트 표시

### 2. GAME START 버튼
- 클릭 시 Mission 1 Stage 1로 진입

---

## 동작 흐름

```
메인 화면
    └── GAME START 클릭 → Mission 1 Stage 1 시작
```

---

## 미정 사항

- 배경 디자인 (정적 이미지 / 애니메이션)
- BGM 및 효과음 여부
- 게임 오버 후 메인 화면 복귀 여부
