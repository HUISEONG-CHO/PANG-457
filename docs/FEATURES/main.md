# 메인 화면 (Main Screen)

## 개요

게임 진입 시 가장 먼저 보이는 화면으로, 게임 시작 및 기본 탐색을 제공한다.

---

## 화면 구성 요소

```
┌─────────────────────────────────────┐
│                                     │
│              PANG                   │  ← 게임 타이틀
│                                     │
│          ▶ 게임 시작                 │  ← 선택된 항목 (포커스 표시)
│            게임 종료                 │
│                                     │
└─────────────────────────────────────┘
```

### 1. 게임 타이틀
- 화면 중앙 상단에 **"PANG"** 텍스트 표시

### 2. 메뉴 항목
| 순서 | 항목 | 동작 |
|------|------|------|
| 0 | 게임 시작 | 게임 화면으로 전환 |
| 1 | 게임 종료 | 종료 확인 후 탭 닫기 |

---

## 키보드 조작

| 키 | 동작 |
|----|------|
| `↑` / `↓` | 메뉴 항목 간 이동 |
| `Enter` | 현재 선택된 항목 실행 |
| 마우스 클릭 | 해당 항목 직접 실행 |

---

## 포커스(선택) 표시

- 현재 선택된 항목 앞에 `▶` 마커 표시
- 선택된 항목은 텍스트 색상 강조 (예: 흰색 → 노란색)
- 비선택 항목은 기본 색상 유지 (예: 회색)
- 최상단에서 `↑` → 최하단 항목으로 순환
- 최하단에서 `↓` → 최상단 항목으로 순환

---

## 동작 흐름

```
메인 화면 진입 → 첫 번째 항목(게임 시작)에 기본 포커스
    ├── ↑ / ↓         → 포커스 이동 (순환)
    ├── Enter          → 현재 항목 실행
    └── 마우스 클릭    → 해당 항목 실행
```

---

## 미정 사항

- 배경 디자인 (정적 이미지 / 애니메이션)
- BGM 및 효과음 여부
- 게임 오버 후 메인 화면 복귀 여부
