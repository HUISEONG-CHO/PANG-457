# 게임 룰 (Game Rules)

## 목표

스테이지에 등장하는 **모든 풍선을 제거**하면 클리어.  
제한 시간 내에 클리어하지 못하거나 풍선에 닿으면 라이프를 잃는다.

---

## 풍선 (Balloon)

### 크기 단계
풍선은 총 4단계 크기로 존재하며, 작살에 맞을 때마다 분열한다.

| 단계 | 크기 | 피격 시 |
|------|------|---------|
| 1단계 | 대 (Large) | 중간 2개로 분열 |
| 2단계 | 중 (Medium) | 소 2개로 분열 |
| 3단계 | 소 (Small) | 극소 2개로 분열 |
| 4단계 | 극소 (Tiny) | 소멸 |

### 이동 방식
- 화면 안에서 **포물선(바운스)** 궤적으로 좌우 이동
- 벽과 바닥에 부딪히면 반대 방향으로 튕김
- 크기가 작을수록 바운스 높이가 낮아지고 이동 속도가 빨라짐

---

## 플레이어

### 조작
| 입력 | 동작 |
|------|------|
| ← → | 좌우 이동 |
| ↑ ↓ | 사다리 오르내리기 |
| 공격 버튼 | 작살 발사 (위 방향) |

- **점프 불가**
- 사다리가 있는 구간에서만 상하 이동 가능

### 사망 조건
- 풍선(어떤 크기든)에 **직접 닿으면** 즉시 사망
- 제한 시간 초과 시 사망
- 라이프를 모두 소진하면 게임 오버

---

## 무기 및 아이템

| 아이템 | 효과 |
|--------|------|
| 기본 작살 | 천장까지 와이어가 유지되며 풍선을 관통 파괴 |
| 더블 작살 | 좌우 두 방향 동시 발사 |
| 파워건 | 연사 가능, 와이어 미형성 |
| 모래시계 | 일정 시간 풍선 이동 정지 |
| 다이너마이트 | 화면 전체 풍선 제거 |

- 아이템은 풍선 격파 시 또는 특정 조건에서 드롭
- 한 번에 하나의 작살만 발사 가능 (기본 상태)

---

## 점수

- 풍선 격파 시 점수 획득
- **큰 풍선일수록 높은 점수**
- 보너스 음식 아이템 획득 시 추가 점수

---

## 스테이지 진행

- 스테이지의 모든 풍선 제거 → 다음 스테이지로 이동
- 각 지역은 3개 스테이지로 구성 (마지막 지역 제외)
- 전체 50개 스테이지
