# PRD - 팡 (PANG) 게임 Mission 1

## 게임 개요

**팡(Pang)** 은 1989년 Mitchell Corporation이 제작한 아케이드 액션 게임이다.  
북미에서는 *Buster Bros.*, 일본에서는 *Pomping World* 라는 이름으로 출시됐다.

플레이어는 전 세계 명소를 습격한 풍선들을 작살로 터뜨리며 지구를 구하는 것이 목표다.  
1~2인 동시 플레이를 지원한다.

---

## 핵심 게임플레이 메커니즘

### 풍선 동작
- 풍선은 화면 안에서 포물선을 그리며 좌우로 튀어 다닌다.
- 작살에 맞으면 **2개의 작은 풍선으로 분열**한다.
- 크기는 총 4단계: 대 → 중 → 소 → 극소 (극소 풍선은 터뜨리면 소멸)
- 스테이지의 모든 풍선을 소멸시키면 클리어

### 플레이어 조작
- 좌우 이동 가능, **점프 불가**
- 사다리를 통해 상하 이동 가능
- 버튼으로 **위쪽 방향 작살** 발사

### 무기 / 아이템
| 아이템 | 효과 |
|--------|------|
| 기본 작살 | 천장에 닿을 때까지 유지되는 와이어형 무기 |
| 더블 작살 | 좌우 두 방향 동시 발사 |
| 파워 작살 | 연사 가능, 단 장애물 미관통 |
| 모래시계 | 일정 시간 풍선 정지 |
| 다이너마이트 | 화면 전체 폭발 |

### 사망 조건
- 풍선에 닿으면 즉시 사망 (라이프 차감)
- 스테이지 제한 시간 초과 시 사망

---

## Mission 1 — 후지산 (Mt. Fuji, Japan)

### 배경
- 일본 후지산을 배경으로 한 게임의 첫 번째 지역
- 총 3개 스테이지로 구성 (아침 → 오후 → 밤 순서로 시간대가 변함)
- 배경 그래픽만 바뀌며 기본 지형 구조는 동일

### 스테이지 구성
- 평탄한 단층 구조의 단순한 지형
- 장애물이 거의 없어 초반 진입 난이도가 낮음
- 대형 풍선 1~2개로 시작, 분열하면서 최대 8개 이상의 극소 풍선이 난무

### 난이도 특징
- 전체 50개 스테이지 중 가장 기초적인 구성
- 풍선 분열 패턴과 이동 동선을 익히기 위한 튜토리얼 성격
- 스테이지가 진행될수록 (아침→밤) 풍선 수 또는 속도가 소폭 증가

---

## 점수 시스템

- 풍선을 터뜨릴 때마다 점수 획득
- 큰 풍선일수록 높은 점수
- 특정 조건에서 보너스 음식 아이템 등장 → 추가 점수
