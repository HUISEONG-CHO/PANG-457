# PLAN.md — Phase별 개발 목표

## 개요

각 Phase는 **실제로 브라우저에서 실행하고 직접 테스트할 수 있는 상태**를 목표로 한다.  
Phase가 끝날 때마다 고객님이 직접 확인하고 피드백을 주시면 다음 Phase로 진행한다.

---

## Phase 1 — 메인 화면 UI

### 목표 기능
- 게임 타이틀 **"PANG"** 표시
- **게임 시작** 버튼 (클릭 시 반응 확인용 alert 또는 콘솔 출력)
- **게임 종료** 버튼 (브라우저 탭 닫기 또는 안내 메시지)
- 기본 레이아웃 및 스타일

### 고객님 테스트 포인트
> "메인 화면이 제대로 보이는가?"

- [ ] 타이틀 "PANG"이 화면에 잘 보이는가
- [ ] 게임 시작 버튼이 클릭되는가
- [ ] 게임 종료 버튼이 동작하는가
- [ ] 전반적인 화면 레이아웃이 자연스러운가

---

## Phase 2 — 화면 전환 (메인 ↔ 게임 씬)

### 목표 기능
- 게임 시작 버튼 클릭 → 게임 화면으로 전환
- 게임 화면: 빈 스테이지 영역 (배경색 또는 이미지)
- 게임 화면에서 메인으로 돌아오는 버튼 (ESC 또는 버튼)

### 고객님 테스트 포인트
> "화면 전환이 자연스러운가?"

- [ ] 시작 버튼 클릭 시 게임 화면으로 전환되는가
- [ ] 게임 화면에서 메인으로 돌아올 수 있는가
- [ ] 전환 시 화면이 깜빡이거나 어색하지 않은가

---

## Phase 3 — 플레이어 캐릭터 이동

### 목표 기능
- 게임 화면에 플레이어 캐릭터 렌더링
- ← → 키로 좌우 이동
- 화면 양 끝(벽) 충돌 처리

### 고객님 테스트 포인트
> "캐릭터가 자연스럽게 움직이는가?"

- [ ] 캐릭터가 화면에 보이는가
- [ ] ← → 키 입력에 즉각 반응하는가
- [ ] 벽에 막혀 화면 밖으로 나가지 않는가
- [ ] 이동 속도가 자연스러운가

---

## Phase 4 — 풍선 등장 및 물리

### 목표 기능
- 대형 풍선 1개 화면에 등장
- 포물선(바운스) 궤적으로 좌우 이동
- 벽과 바닥에 튕기는 물리 처리

### 고객님 테스트 포인트
> "풍선이 자연스럽게 튀어 다니는가?"

- [ ] 풍선이 화면에 보이는가
- [ ] 벽과 바닥에 자연스럽게 튕기는가
- [ ] 포물선 궤적이 원작과 유사하게 느껴지는가

---

## Phase 5 — 작살 발사

### 목표 기능
- 공격 키(Space 또는 Z)로 위 방향 작살 발사
- 작살이 천장에 닿으면 소멸
- 발사 중 추가 발사 불가 (1발 제한)

### 고객님 테스트 포인트
> "작살이 올바르게 발사되고 사라지는가?"

- [ ] 공격 키를 누르면 작살이 위로 발사되는가
- [ ] 천장에 닿으면 작살이 사라지는가
- [ ] 작살이 존재하는 동안 추가 발사가 막히는가

---

## Phase 6 — 풍선 분열 + 플레이어 충돌

### 목표 기능
- 작살이 풍선에 명중 → 2개의 작은 풍선으로 분열
- 4단계(대→중→소→극소) 분열 후 소멸
- 플레이어가 풍선에 닿으면 사망 처리 (화면 표시)

### 고객님 테스트 포인트
> "게임의 핵심 규칙이 동작하는가?"

- [ ] 작살로 풍선을 맞추면 2개로 쪼개지는가
- [ ] 4단계 분열 후 마지막 풍선이 소멸하는가
- [ ] 풍선에 닿으면 사망 연출이 나오는가

---

## Phase 7 — 라이프 시스템 + 게임 오버

### 목표 기능
- 라이프 3개 (HUD에 표시)
- 사망 시 라이프 1 차감 후 재시작
- 라이프 0 → 게임 오버 화면 표시
- 게임 오버 후 메인 화면으로 복귀

### 고객님 테스트 포인트
> "목숨 시스템이 올바르게 동작하는가?"

- [ ] 사망할 때마다 라이프가 줄어드는가
- [ ] 라이프가 0이 되면 게임 오버 화면이 뜨는가
- [ ] 게임 오버 후 메인 화면으로 돌아오는가

---

## Phase 8 — 스테이지 클리어 + Mission 1 구성

### 목표 기능
- 모든 풍선 제거 → 스테이지 클리어 연출 및 다음 스테이지 진행
- Mission 1의 3개 스테이지 (Stage 1 → 2 → 3) 순서 진행
- Stage별 풍선 수 구성 반영
- Mission 1 완주 시 미션 클리어 화면

### 고객님 테스트 포인트
> "Mission 1을 처음부터 끝까지 플레이할 수 있는가?"

- [ ] 풍선을 모두 없애면 다음 스테이지로 넘어가는가
- [ ] Stage 1 → 2 → 3 순서로 진행되는가
- [ ] Stage 3 클리어 후 미션 완료 화면이 뜨는가
- [ ] 스테이지마다 풍선 수가 다른가

---

## Phase 9 — 점수 시스템 + HUD 완성

### 목표 기능
- 풍선 격파 시 크기별 점수 획득
- 점수 HUD 표시 (현재 점수 / 라이프)
- 게임 오버 시 최종 점수 표시

### 고객님 테스트 포인트
> "점수가 올바르게 쌓이고 표시되는가?"

- [ ] 풍선을 터뜨릴 때마다 점수가 올라가는가
- [ ] 큰 풍선과 작은 풍선의 점수가 다른가
- [ ] 게임 오버 화면에서 최종 점수를 확인할 수 있는가

---

## Phase 10 — 아이템

### 목표 기능
- 풍선 격파 시 랜덤 아이템 드롭
- 더블 작살 / 모래시계 / 다이너마이트 효과 구현

### 고객님 테스트 포인트
> "아이템이 드롭되고 효과가 정상 동작하는가?"

- [ ] 아이템이 화면에 떨어지는가
- [ ] 아이템을 먹으면 해당 효과가 발동하는가
- [ ] 모래시계 사용 시 풍선이 멈추는가
- [ ] 다이너마이트 사용 시 화면의 풍선이 모두 제거되는가

---

## Phase 11 — 배경 + 사운드 + 연출

### 목표 기능
- Mission 1 배경 이미지 적용 (아침 / 오후 / 밤 시간대)
- 배경 BGM 및 효과음 (발사, 분열, 사망, 클리어)
- 캐릭터·풍선 애니메이션 개선

### 고객님 테스트 포인트
> "완성된 게임처럼 느껴지는가?"

- [ ] 스테이지마다 배경이 다르게 보이는가
- [ ] 사운드가 게임 상황에 맞게 재생되는가
- [ ] 전반적인 비주얼과 연출이 자연스러운가
